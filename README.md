# Oasis Infobyte Data Analytics Internship

This repository contains the projects completed as part of my Data Analytics Internship at Oasis Infobyte.

## Tasks Completed

### Task 1: Exploratory Data Analysis (EDA) on Retail Sales Data
- Performed data exploration and visualization.
- Identified sales patterns and business insights.
- Used Python, Pandas, Matplotlib, and Seaborn.

### Task 2: Data Cleaning
- Cleaned the Titanic dataset.
- Handled missing values and duplicate records.
- Detected outliers and prepared the dataset for analysis.

### Task 3: Sentiment Analysis
- Built a sentiment analysis model using Amazon product reviews.
- Applied TF-IDF vectorization and a Naive Bayes classifier.
- Evaluated the model using accuracy, a classification report, and a confusion matrix.
